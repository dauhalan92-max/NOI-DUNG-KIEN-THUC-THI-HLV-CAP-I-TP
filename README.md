# HLV Cấp I - Hội LHTN Việt Nam TP Đà Nẵng

## Đưa lên GitHub Pages
1. Giải nén file ZIP.
2. Tạo repository mới trên GitHub.
3. Upload toàn bộ file trong thư mục này lên repository.
4. Vào Settings → Pages → Deploy from a branch → chọn `main` và `/root` → Save.
5. Mở link GitHub Pages được cấp.

## Cài như app điện thoại
Mở link web bằng Safari/Chrome → Chia sẻ → Thêm vào Màn hình chính.

## Ghi chú
- 6 game được giữ riêng theo từng file.
- Trang chủ có hiệu ứng động, âm thanh, chọn theo thứ tự/xáo trộn.
- Sơ cấp cứu giữ hình minh họa trong câu hỏi.
- Không giới hạn thời gian trả lời.
