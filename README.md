- Khôi phục 6 game từ file gốc đã kiểm tra.

- Chế độ xáo trộn chỉ xáo trộn thứ tự câu hỏi, không xáo trộn đáp án trong câu.
- Không dùng dữ liệu cũ trong trình duyệt khi mở game từ trang chủ.
